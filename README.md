# Gir Lion Genomic Diversity Analysis

## Study
Comparative whole-genome analysis of Asiatic lion genomic diversity

## Authors
- Akshay Krishnan Pushparaj
- Malarmathi Muthukumar
Dr. N.G.P. Arts and Science College, Coimbatore, India

## Publication
Pushparaj AK, Muthukumar M (2026). Genomic diversity deficit,
elevated inbreeding, and conservation implications for the
endangered Asiatic lion (Panthera leo persica): A comparative
whole-genome analysis. Evolutionary Applications.

## Data
All genomic data from published sources:
- Asiatic lion: NCBI BioProject PRJNA379375 (Mitra et al. 2019)
- African lion: Cho et al. 2013 (Nature Communications)
- Additional felids: See manuscript Table 1

## Requirements
- Python 3.12+
- pandas, numpy, scipy, matplotlib, seaborn

## Usage
python analysis.py
